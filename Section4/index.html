HTML & CSS Basic Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content, structure and meaning. You don't use it for styling. Use CSS for that.

  • HTML uses elements to describe (annotate) content
  • HTML elements typically have an opening tag, content and then closing tag
  • You can also have void (empty)
  • You can also configure elements with attributes
  • There's a long list of availablee elements but you'll gain experience over time, no worries

Learn more about all available HTML elements on the MDN HTML element reference.

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content

  • Styles are assigned via property-value pairs
  • You can assign styles via the "style" attributes
  • To avoid code duplication, you typically use global styles(via the "style" element) instead
  • Alternatively, you can work with external stylesheet files which you can "link" to
  • When working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood

Learn more aobut all availabel CSS properties and values on the MDN CSS property reference.
